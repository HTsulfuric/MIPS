# 0xabcdef12の格納

- メモリが4byte区切り & 16進数表現は1桁の表現に4bit(1/2バイト)を使用することを覚えてれば

## Little Endianの場合

- メモリアドレスの低い方に、格納したい値の下位バイトから順に格納

| アドレス |  値  |
| :------: | :--: |
|   0x00   | 0x12 |
|   0x01   | 0xef |
|   0x02   | 0xcd |
|   0x03   | 0xab |

## Big Endianの場合

- メモリアドレスの低い方に、格納したい値の上位バイトから順に格納

| アドレス |  値  |
| :------: | :--: |
|   0x00   | 0xab |
|   0x01   | 0xcd |
|   0x02   | 0xef |
|   0x03   | 0x12 |
